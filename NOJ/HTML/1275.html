1275.判断白方当前有几个符合必吃规则的棋子

时限：1000ms 内存限制：10000K  总时限：3000ms

描述
给定一个对阵棋局，编写一个程序求在当前状态下白方有几颗棋子符合“必吃规则”。（必吃规则在第一题中详细介绍过，在此不做重复）
输入
先输入一个8*8的整数矩阵表示棋盘的当前状态
输出
输出白方棋子中符合“必吃规则”的棋子个数。
输入样例
1 0 1 0 1 0 1 3
3 1 6 1 0 1 0 1
1 3 1 3 1 0 1 3
3 1 0 1 0 1 0 1
1 0 1 0 1 0 1 5
5 1 0 1 3 1 3 1
1 5 1 5 1 0 1 5
5 1 5 1 5 1 5 1
输出样例
3

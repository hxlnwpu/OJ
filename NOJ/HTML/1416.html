1416.简单计算

时限：1000ms 内存限制：10000K  总时限：3000ms

描述
示例程序
输入两个整数，若第一个数大于第二个数，输出它们的和；否则输出它们的积。
标准程序如下：
#include <stdio.h>

int main()
{
int a, b;

scanf("%d%d", &a, &b);

if(a > b)
{
printf("%d\n", a + b);
}
else
{
printf("%d\n", a * b);
}

return 0;
}
本题要求输入两个整数，若第一个数大于0，输出第二个数；否则输出它们的和。
输入
输入两个整数
输出
若第一个数大于0，输出第二个数；否则输出它们的和。
输入样例
3 5
输出样例
5

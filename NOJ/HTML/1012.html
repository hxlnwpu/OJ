1012.冒泡排序(bubble sort)

时限：1000ms 内存限制：10000K  总时限：3000ms

描述
给定n（1<N<1000）个整数，每个整数的大小都在long型整数的表示范围内，请对这N个数进行排序，结果按从小到大的顺序输出。
Given n (1<N<1000) integers, the result output in the order from small to large.
输入
第一行从标准输入输入一个整数n(不超过20)，在接下来的n行，每行一个整数。
Input a positive integer n(no more then 20) first, in the next n lines, each line is an integer.
输出
将这n个整数排序后，从小到大输出到标准输出，每个整数占一行。
These n integers sorted from small to large and output, one line per integer.
输入样例
5
5
4
3
2
1
输出样例
1
2
3
4
5

1545.合并多项式

时限：1000ms 内存限制：10000K  总时限：3000ms

描述
输入两个多项式，如3x^2+2x+1 和 -4x^2+1,合并两个多项式结果为：-x^2+2x+2
输入
按幂指数从高到低输入两个多项式，首先输入第一个多项式的幂指数个数n（包括幂指数是0的），以下每一行是多项式的系数和幂指数，第二个多项式的输入类似。注意：最高幂指数是10，多项式之间默认是加，系数可以是负值，系数如果是0，则不输入
输出
按照幂指数从高到低输出多项式，如：-x^2+2x+2，系数为零的不输出，如果结果是0的话输出0。
输入样例
3
3 2
2 1
1 0
2
-4 2
1 0
输出样例
-x^2+2x+2

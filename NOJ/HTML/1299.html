1299.悔棋的实现

时限：1000ms 内存限制：10000K  总时限：3000ms

描述
完成悔棋功能，本功能采用堆栈实现，建立一个节点该节点结构如下：
int lastx；存放移动棋子的原来位置横坐标
int lasty；存放移动棋子的原来位置纵坐标
int last；存放移动棋子的标志符
int futurex，futurey；存放棋子将要移动到的位置的坐标
int future；存放移动棋子将要移动到的位置的标志符
悔棋一步的操作就是，将栈顶节点取出，回复成操作前的矩阵。
如果建立两个栈则可以实现悔棋，恢复原状的功能。
考虑更合适的动态局面评估策略，选择合适的算法以使电脑水平更高些。比如可以考虑下一步操作后的态势。或者对局面进行判断时，棋子权值随棋盘上棋子的数量和位置的不同有不同的权值。比如一个兵如果其位于将的周围，这个兵的威胁性或者说是权值可能比车都要大。再比如在棋子数量很少的情况下，炮的权值要远远小于马。同学们课下考虑实现

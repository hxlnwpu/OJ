1415.算12（三）（count 12 （三））

时限：1000ms 内存限制：10000K  总时限：3000ms

描述
输入3个正整数a，b，c，计算表达式（（a&b）&c）的值，（&为加减乘三种运算之一），能否得到12。
Input 3 positive integers a, b, c, Calculate the value of the expression ((a&b)&c),(&is one of the three operator: add, subtract, multiply), can I able to get 12.

输入
输入3个正整数，
Input 3 positive integer,

输出
如果能得到12，则输出Yes，否则输出No。
If you can get 12, then output Yes, otherwise output the No.

输入样例
2 2 3
输出样例
Yes

1273.给定坐标的白兵下一跳可能的移动路径有几条

时限：1000ms 内存限制：10000K  总时限：3000ms

描述
给定一个白兵的坐标，编写一个程序求这个白兵有几种可能的移动路径（符合必吃规则时必须吃棋）。
输入
先输入一个8*8的整数矩阵表示棋盘的当前状态，再输入两个为整数m，n，表示白兵所在的行和列，其中0<=m，n<=7。
输出
一个整数，表示可能的移动路径的条数。
输入样例
1 3 1 3 1 3 1 3
3 1 0 1 0 1 0 1
1 3 1 3 1 0 1 3
3 1 0 1 0 1 0 1
1 0 1 0 1 0 1 5
5 1 0 1 3 1 3 1
1 5 1 5 1 0 1 5
5 1 5 1 5 1 5 1
6
3
输出样例
1

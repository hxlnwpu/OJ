1160.怪物和兔子的故事

时限：1000ms 内存限制：10000K  总时限：3000ms

描述
你来到了一个奇怪的小镇，镇上有M只怪物和N只小兔子


镇上的居住者们遇到的时候会有如下反应：

1.当两只兔子遇到时，什么都不发生
2.当一只怪兽遇到一直兔子时，怪兽把兔子吃掉了
3.当两只怪兽相遇时，他们开始打架，然后他们两个都死了
4.当你遇到一只兔子时，你可以选择把它杀掉或者不杀，而兔子不会对你造成任何危害。
5.当你遇到一只怪兽时， 怪兽会马上把你吃掉


居民们在镇上随机游荡，我们作以下假设

1.碰面不会同时发生
2.每次只有两个生物会相遇
3.这两个生物从镇上现有生物中随机选出，每个生物被选的几率是一样的（包括你）


你会胜利如果某个时刻你不再会被杀死，在你胜利或死之前你不允许出镇子


你需要做的是计算出你活下来的概率
输入
两个整数 M和N (N,M<=1000)
输出
你活下来的概率,精确到10e-9
提示：
输出语句为：
printf("%.9lf\n",Q);
输入样例
0 0
输出样例
1.000000000

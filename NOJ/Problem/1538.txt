    1538.判断二叉树是不是完全二叉树
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    先序输入二叉树数据，中序输入二叉树数据，重建二叉树，判断其是不是完全二叉树。
   
    输入
   
    第一行输入二叉树先序遍历序列，第二行输入二叉树的中序遍历序列，0表示每一行的输入结束。
   
    输出
   
    判断是不是完全二叉树，1表示是，0表示不是。
   
    输入样例
   
    A B D E F C 0
    
    D B E F A C 0
    
    输出样例
   
    0

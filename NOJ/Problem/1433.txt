    1433.判断一个运算是否有幺元
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    给定一个集合A={0, 1, 2, ..., n-1}，及A上的运算*的运算表，判断该运算是否有幺元。
   
    输入
   
    先输入一个小于等于20的正整数n，接下来输入一个n行n列的由0到n-1的数构成的矩阵表示这个运算的运算表。
    
    输出
   
    如果该运算存在幺元则输出“Yes”，否则输出“No”。
    
    输入样例
   
    4
    
    0 1 2 3
    
    1 3 1 3
    
    2 0 2 0
    
    3 1 3 1
    
    输出样例
   
    Yes

    1535.求二叉树中两个节点的最低公共祖先节点
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    先序输入一棵二叉树数据，建立二叉树，再输入二叉树的两个节点，求这两个节点的最低公共祖先节点。
   
    输入
   
    先序输入一棵二叉树的数据,#表示空节点。然后输入两个节点值
   
    输出
   
    输出这两个节点的最低公共祖先节点。
   
    输入样例
   
    A B D # # E # F # # C # #
    
    F C
    
    输出样例
   
    A

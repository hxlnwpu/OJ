    1408.算12（一）（count 12）
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    输入三个正整数，把他们加起来，判断结果是否等于12。
    
    Input 3 positive integers, add the 3 number, Judge whether the result is equal to 12
   
    输入
   
    输入三个正整数，
    
    Input 3 positive integers,
   
    输出
   
    把他们加起来，如果结果是12，则输出Yes，否则输出No。
    
    add the 3 number, If the result is 12, then the output Yes, otherwise output No.
   
    输入样例
   
    2 6 4
    
    输出样例
   
    Yes

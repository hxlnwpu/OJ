    1607.勾股数（二）前两位相差1的勾股数
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    佳佳在编写勾股数（一）的程序的时候，出了点错误，当他输入100的时候，运行时结果是20 21 29，他验证了一下是勾股数，但是他确实不知道程序哪里出了错误，你能用单步调试的方法帮他找出错误在哪一行吗？更进一步你能找出n以内的所有的前两个数相差是1的勾股数吗？
    
    实际上他的程序在n等于100的时候出现了非常奇怪的结果，
    
    #include<iostream>
    
    using namespace std;
    
    int main()
    
    {
    
    int a, b, n;
    
    cin >> n;
    
    for(a = 3; a <= n; a++)
    
    {
    
    for(b = 6; b <= n; b++)
    
    {
    
    if(a * a + (a + 1) * (a + 1) == b * b)
    
    {
    
    cout << a << " " << a + 1 << " " << b << endl;
    
    }
    
    }
    
    }
    
    }
    
    输入
   
    输入一个正整数n，
   
    输出
   
    输出n以内的所有的前两个数相差是1的勾股数。
   
    输入样例
   
    200
    
    输出样例
   
    3 4 5
    
    20 21 29
    
    119 120 169

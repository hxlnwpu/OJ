    1104.复制字符串
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    编写一个函数void strcopy(char *s,char *d)，函数的功能是将d所指向的字符串复制到s所指向的字符型数组中(不能使用strcpy函数)。
   
    输入
   
    在主函数中输入一个字符串。
   
    输出
   
    调用函数复制到另一个字符串中且打印出来。
   
    输入样例
   
    This is a c program
    
    输出样例
   
    This is a c program

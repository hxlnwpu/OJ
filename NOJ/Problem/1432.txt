    1432.判断一个双射函数是否为同构函数
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    已知两个代数系统，它们的载体中都有n个元素，还有一个从第一个代数系统的载体到第二个代数系统载体的双射函数h，判断h是否为同构函数。
   
    输入
   
    先输入一个正整数n，接下来输入两个矩阵表示两个运算的运算表，最后n个数表示函数h。
   
    输出
   
    若h是同构函数，则输出“Yes”，否则输出No。
   
    输入样例
   
    2
    
    0 1
    
    0 1
    
    1 0
    
    0 0
    
    0
    
    1
    
    输出样例
   
    No

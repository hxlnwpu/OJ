    1249.天平称球(weigh ball using Balance)
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    已知有n个外观一样的球，其中的n - 1个一样重，1个偏重，求最少使用天平称几次才能够找出那个偏重的球？（假设天平没有砝码）
    
    There are n balls that their are the same outward appearance, in them the n - 1 has the same weight, and the other one is Relatively heavy. calculate minimum number of times you will use the balance to find the  Relatively heavy one.
   
    输入
   
    一个正整数n(2 <= n <= 1000000)，表示球的总个数
    
    A positive integer n (2 < = n < = 1000000), n is the total number of the balls.
   
    输出
   
    最少称的次数
    
    The minimum number of weighings
   
    输入样例
   
    2
    
    输出样例
   
    1

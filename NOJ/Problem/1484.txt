    1484.稀疏矩阵转置
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    输出稀疏矩阵的转置矩阵。（行列均不大于20）
   
    输入
   
    输入两个正整数n和m,为矩阵的行数和列数,然后输入矩阵三元组，（0 0 0）表示结束输入。
   
    输出
   
    转置后的矩阵。
   
    输入样例
   
    4 4
    
    1 1 1
    
    2 1 2
    
    3 2 3
    
    0 0 0
    
    输出样例
   
    1 1 1
    
    1 2 2
    
    2 3 3

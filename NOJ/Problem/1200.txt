    1200.判断三角形是否为直角三角形
   
    时限：100ms 内存限制：10000K  总时限：1000ms
   
    描述
   
    给定平面直角坐标系中的三个点的坐标，判断是否能构成直角三角形。
   
    输入
   
    三个点的坐标，数据都在-100000到100000之间。
   
    输出
   
    一个整数，当可以构成直角三角形的时候输出1，否则输出0。
   
    输入样例
   
    0 0
    
    3 0
    
    0 8
    
    输出样例
   
    1
    
    提示
   
    输出语句：printf("%d\n",1);

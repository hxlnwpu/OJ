    1270.统计双方剩余棋子之差？（红方-白方）
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    国际象棋简介：
    
    国际跳棋的棋盘是由64个黑白相间的小方格组成的方形棋盘。棋子为圆盘形状,共24枚,分黑白两色。对局开始时,每一方有12枚棋子，摆成如图(图I) 的样子。单子的盘角置于左边，双子的盘角置于右边。
    
    兵的走法和吃子规则：
    
    一、兵每次只能向前方斜走一格至下一排的一个空格。兵移动时，只能前进，不能后退。
    
    二、当己方的一只兵，在它所处的斜线上遇到对方的一只与它相邻的棋子时，如果对方的这只棋子后面是个空格，则己方的兵必须跳过对方的这只棋子，并占据这个空格，同时把对方的这只棋子必须立即从棋盘上拿走，这就是兵的吃子过程。兵可以向前方也可以向后方吃掉对方的棋子。
    
    三、兵在吃子的过程中，当它遇到对方另一只与它相邻的棋子，并且对方的这只棋子后面也是个空格，那么这只兵必须再次跳过第二只，第三只以及更多可被吃掉的棋子，最终占据到最后被吃掉的棋子后面的那个空格子。被吃掉的棋子按照吃子上升或下降的顺序逐个依次从棋盘上拿走。
    
    注：兵在连续吃子的过程中，允许多次跳跃同一个空格，但禁止第二次跳跃对方的同一个棋子。在连续吃子的过程中，禁止跳跃己方的棋子。
    
    王的走法和吃子规则：
    
    一、当一个兵走到并停在对方底线上的任何一个升变格内时，它将升变为王如图(图II)，并在这只兵的上面叠上另一只同样颜色（即同一方）的兵来表示。王可以在它所在的斜线上，向前方和向后方移动任意格数的距离。
    
    二、当己方的一只王在相同的斜线上遇到对方的一只与它相邻或不相邻的棋子时，如果对方的这只棋子后面有一个或多个空格，那么这只王必须跳过对方的这只棋子并占据它后面的任意一个空格，同时对方的这只棋子必须立即从棋盘上拿走。这就是王的吃子过程。王可以向前方也可以向后方吃掉对方的棋子。
    
    三、王在吃子的过程中，在相同的斜线或与它垂直的另一条斜线上遇到对方的另一只与它相邻或不相邻的棋子时，如果对方的这只棋子后面有一个或多个空格，那么这只王棋必须跳过对方的第二只棋子，第三只以及更多可被吃掉的棋子，最终占据到与最后被吃掉的棋子同一条斜线上后面任意的一个空格，被吃掉的棋子按照吃子上升或下降的顺序逐个依次从棋盘上拿走。这就是王连续吃子的整个过程。
    
    注： 王在连续吃子的过程中，允许多次跳跃同一个空格，但禁止第二次跳跃对方的同一个棋子。在连续吃子的过程中，禁止跳跃己方的棋子。
    
    必吃规则：
    
    王与兵有跳吃对方棋子的机会时,不管对自己是否有利，规则要求必须选择吃掉对方棋子的走法。
    
    自选规则:
    
    王与兵在跳吃对方的棋子时,如果棋盘上出现一种以上的吃子路线的话，吃子方可以自由选择其中的任何一种吃子的路线。
    
    升变规则：
    
    当一个兵走到并停在对方底线上的任何一个升变格内时，它将必须升变为王。
    
    当一个兵只能路过在对方底线上的任何一个升变格，而不能停留在那个升变格内时，也可以而且必须升变为王。并且继续吃下去的时候，必须根据王棋的走法来跳。
    
    在表示该棋盘对阵状态时，用8*8的二维数组表示：空格用“0”表示，墙用“1”表示，红方的兵用“3”表示，红方的王用“4”表示，白方的兵用“5”表示，白方的王用“6”表示。约定：棋局开始状态为红方在上，白方在下，即红兵只能向下走，白兵只能向上走。
    
    给定一个对阵的棋盘，编写一个程序求这个棋盘上双方剩余棋子之差？（用红方棋子数减去白方棋子数）。
   
    输入
   
    输入一个8*8的整数矩阵表示棋盘的当前双方对阵状态
   
    输出
   
    一个整数，表示双方剩余棋子之差（用红方棋子数减去白方棋子数）。
   
    输入样例
   
    1 3 1 3 1 3 1 3
    
    3 1 0 1 0 1 0 1
    
    1 3 1 3 1 0 1 3
    
    3 1 0 1 0 1 0 1
    
    1 0 1 0 1 0 1 5
    
    5 1 0 1 3 1 3 1
    
    1 5 1 5 1 0 1 5
    
    5 1 5 1 5 1 5 1
   
    输出样例
   
    2

    1025.字符的ASCII码(ASCII code)
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    输入一个字母，输出其ASCII码值。
    
    Input a  character, output the value of the ASCII code.
   
    输入
   
    一个字母
    
    Input a  character,
   
    输出
   
    输出这个字母的ASCII码，最后输出一回车
    
    Output ASCII code of the letter
   
    输入样例
   
    a
    
    输出样例
   
    97

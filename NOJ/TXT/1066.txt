    1066.字符串排序(string sort)
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    从键盘输入4个字符串（长度小于20），存入二维字符串数组中。然后对它们进行排序（假设由小到大顺序），最后输出排序后的4个字符串。
    
    Input 4 strings (length no more than 20), stored into a two-dimensional string array, sort these strings (ascending), output the sorted strings.
   
    输入
   
    输入4个字符串（长度小于20），每个单独一行。
    
    Enter the 4 string (length less than 20), each a separate line.
   
    输出
   
    输出排序后的4个字符串，每个单独一行。
    
    Output the sorted strings,each a separate line
   
    输入样例
   
    calrfeoiiojdsf
    
    jdaoejffdsf
    
    afxeezc
    
    asdfeasdf
    
    输出样例
   
    afxeezc
    
    asdfeasdf
    
    calrfeoiiojdsf
    
    jdaoejffdsf
    
    提示
   
    字符串比较可以用strcmp函数实现，排序方法可以是选择法或冒泡法。

    1077.着色
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    对于每个象素点，用RGB三色的值代表它，用（R，G，B）来表示，其中R、G、B的值都为0-255（包括边界）。任两个点的距离用下面的这个公式：
    
    输入
   
    先给出16个点的RGB值，再给出若干个点的RGB值，对这些点请找出在前面16个点中离它们最近的点是哪一个。以三个-1代表测试结束。
   
    输出
   
    对每一个点，输出离他们最近的点，例如：点（77，79，134）最近的点为点（128，128，128），则输出：
    
    （77，79，134） maps to  (128,128,128)
    
    输入样例
   
    0 0 0
    
    255 255 255
    
    0 0 1
    
    1 1 1
    
    128 0 0
    
    0 128 0
    
    128 128 0
    
    0 0 128
    
    126 168 9
    
    35 86 34
    
    133 41 193
    
    128 0 128
    
    0 128 128
    
    128 128 128
    
    255 0 0
    
    0 1 0
    
    0 0 0
    
    255 255 255
    
    253 254 255
    
    77 79 134
    
    81 218 0
    
    -1 -1 -1
    
    输出样例
   
    (0,0,0) maps to (0,0,0)
    
    (255,255,255) maps to (255,255,255)
    
    (253,254,255) maps to (255,255,255)
    
    (77,79,134) maps to (128,128,128)
    
    (81,218,0) maps to (126,168,9)

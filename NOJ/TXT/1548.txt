    1548.求图的连通分量的个数
   
    时限：1000ms 内存限制：10000K  总时限：3000ms
   
    描述
   
    求图的连通分量的个数。
   
    输入
   
    先输入一个小于等于100的正整数n，表示图中结点的个数，接下来的n行每行n个整数（0或1）表示图的邻接矩阵，
   
    输出
   
    输出图的连通分量的个数。
   
    输入样例
   
    3
    
    0 0 0
    
    0 0 1
    
    0 1 0
    
    输出样例
   
    2
